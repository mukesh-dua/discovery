# Project Templates

## Overview

You can create Microsoft Discovery Projects in two ways: using pre-built template agents or creating custom agents. Templates offer ready-to-use agents with functionality that's ideal for testing platform features and scenarios. 

While some templates include specialized tools and knowledge capabilities, you may need to build custom agents to fully address your specific requirements.

## Published templates

### Hello Discovery
The Hello Discovery Template provides an interactive learning experience powered by specialized AI agents that answer questions about the Microsoft Discovery platform. The agents are not equipped with scientific tools and knowledge which might be required for some use-cases.

#### How it works

Ask any question about Discovery platform, and the team of agents will route it to the right expert, gather information, and provide a comprehensive answer based on official documentation.

#### Included agents

- **RouterAgent** - Analyzes your question and directs it to the appropriate expert agent
- **AgentExpert** - Answers questions about Discovery Agents, including types, configurations, and workflow orchestration
- **SuperComputerExpert** - Explains Discovery Supercomputer, nodepools, HPC capabilities, scaling, and infrastructure
- **ToolsExpert** - Covers Discovery Tools, types, deployment, and integration
- **SummarizerAgent** - Compiles expert responses into a clear, comprehensive final answer

#### What you can ask

Try some of these sample prompts -

- How do Discovery Agents work?
- What are nodepools and how do they scale?
- How do I deploy tools to the platform?
- What's the difference between workflow agents and specialized research agents?
- How does storage integration work with the Supercomputer?

Each expert provides accurate, detailed answers strictly from their knowledge domain, ensuring you get reliable information about the Discovery platform.
