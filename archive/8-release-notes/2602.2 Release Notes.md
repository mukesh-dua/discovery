# Microsoft Discovery Release Updates 2602.2

**Release Date:** February 27, 2026

---

## Summary
Release **2602.2** focuses on continued platform stabilization, reliability improvements, and incremental refinements across the Microsoft Discovery experience.

---

## What's New (Key Product Features)


---

[**Microsoft Discovery Studio v2**](https://github.com/microsoft/discovery/blob/main/4-how-to/10-discovery-studio-preview-experience/a--preview-experience.md)

**Improvements and Enhancements**
- Incremental **Studio v2** refinements, including:
  - Theming updates
  - Welcome Page redesign

---

[**Microsoft Discovery Supercomputer**](https://github.com/microsoft/discovery/blob/main/3-concepts/supercomputer.md)

**Improvements and Enhancements**
- Added support for **Azure VM SKUs using AMD 5th Gen EPYC processors**
- Users can now select **more than twenty (20) nodes** within a given nodepool directly from the user interface
