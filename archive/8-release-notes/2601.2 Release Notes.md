# Microsoft Discovery Release Updates 2601.2

### Release Date
Feb 6 2026

## Summary
**Release 2601.2** delivers enhancements and improvements across multiple features. 
Fundamentals work also continued improving platform stability and scalability.


## What's New (Key Product Features)
[**Discovery Studio v2**](https://github.com/microsoft/discovery/blob/main/4-how-to/10-discovery-studio-preview-experience/a--preview-experience.md)

#### Improvements and Enhancements
* Improved one-click project creation experience with option to auto-create data containers
* Changed "refinement" terminology back to "investigations" in the UI based on user feedback
* Completed several bug fixes and UI enhancements
* Increased UX integration test coverage
