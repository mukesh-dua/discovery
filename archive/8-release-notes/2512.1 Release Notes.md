# Microsoft Discovery Release Updates 2512.1

### Release Date
Dec 22 2025

## Summary
**Release 2512.1** stabilizes core Microsoft Discovery Platform workflows, targeted reliability fixes across Bookshelf indexing, Discovery Studio improvements, ACA resource sizing, and deployment pipelines. 

## What's New (Key Product Features)
[**Discovery Studio v2**](https://github.com/microsoft/discovery/blob/main/4-how-to/10-discovery-studio-preview-experience/a--preview-experience.md)

* Discovery Studio v2 is now the default experience when opening a project from Studio
* Project resources pane to view agents, tools, KBs, and data containers associated with a project
* Classic project experience can be accessed from the workspace view by clicking on the ellipsis

**Governance and Reliability**
* Infrastructure to ingest logs from MRG resources into Log Analytics workspace

**East US Deployments**

Prior to 2512.1 Release, Discovery workspaces deployed in East US were mapped to Azure AI Foundry resources in East US 2.  

As of this release, all new Discovery workspaces created in East US will now map to Foundry resources in the same region.
* While Discovery has engineered compatibility for a mixed region deployment, it is not recommended to maintain a mixed environment.

* Creating a new workspace in East US is recommended in place of current workspaces deployed to this region.
* Please reference the refreshed region quota requirements as part of this update [here](https://github.com/microsoft/discovery/blob/main/4-how-to/2-onboarding-experience/b--quota-reservations.md)
