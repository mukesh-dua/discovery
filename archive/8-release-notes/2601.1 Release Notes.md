# Microsoft Discovery Release Updates 2601.1

### Release Date
Jan 16 2026

## Summary
**Release 2601.1 (1.1)** delivers enhancements and improvements across multiple features. 
Fundamentals work also continued improving platform stability and scalability.


## What's New (Key Product Features)
[**Discovery Studio v2**](https://github.com/microsoft/discovery/blob/main/4-how-to/10-discovery-studio-preview-experience/a--preview-experience.md)

#### Improvements and Enhancements
* Project creation flow enhanced with a one‑click experience that lets users add sample templated agents right from the start
* Added [Discovery Engine](https://github.com/microsoft/discovery/blob/main/3-concepts/engine.md) dashboard for users to visualize Tasks in a Refinement
* Added support for exporting debug data, an easier way for users to share their data, chats and logs for better debugging
