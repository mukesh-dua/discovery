## Silicon Development Use Cases
| ID | Use Case | Description |
|-|-|-|
|1.  | **RTL Code Generation** | Create RTL code based on a complete specification document. |
|2.  | **Linting** | Correct syntax and code correctness of RTL code |
|3.  | **Testbench Generation** | Create testbench based on spec validation requirements and RTL code. |
|4.  | **Simulation** | Verify code correctness at RTL, gate-level, and post-layout verification using generated testbenches and provide fixes |
|5.  | **Test Insertion** | Insert test structures based on testing parameters in the spec |
|6.  | **Formal Verification** | Create formal properties for validation |
|7.  | **Intent to Spec** | Develop Specifications based on customer intent and associated collateral |
|8.  | **Spec to RTL** | Develop golden RTL based on specifications documentation and supporting collateral |
|9.  | **RTL to GDSII** | Process RTL code through layout and validation (logic and physical) to generating GDSII |
