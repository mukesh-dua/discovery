# Example Input Files for AmberTools Agent

## Files

| `alanine-dipeptide.pdb` | Peptide | 22 | ACE-ALA-NME minimal test system | Generated by tleap |

## Quick Tests (< 5 minutes)

### Test 1: Alanine Dipeptide Minimization
Smallest possible system. Tests: tleap, sander minimization.
```
Input: alanine-dipeptide.pdb
Expected: system.prmtop, system.inpcrd, min.mdout with converged energy
```

### Test 2: Small Molecule Parameterization
Tests: antechamber, parmchk2, tleap for ligands.
```
Input: aspirin.sdf
Expected: ligand.mol2, ligand.frcmod, ligand.prmtop
```

### Test 3: Trp-cage Preparation + Short MD
Realistic protein system. Tests: pdb4amber, tleap, sander (minimize + 100 steps MD).
```
Input: 1l2y.pdb
Note: This is an NMR structure with 38 models. Extract MODEL 1 first:
  grep -A 99999 "^MODEL        1" 1l2y.pdb | grep -B 99999 "^ENDMDL" | head -n -1 > 1l2y_model1.pdb
```
